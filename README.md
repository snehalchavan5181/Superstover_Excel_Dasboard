# Superstore Excel Dashboard Project

## Overview
Interactive Excel Dashboard with Pivot Tables, Charts, and Slicers.

## Features
- Line Chart: Monthly Sales Trend
- Column Chart: Top Products Sales
- Pie Chart: Sales by Region
- Slicers: Filter by Region / Sales
- Auto-update charts when data changes

## Usage
Open `Superstore_Dashboard.xlsx` in Excel and use slicers to filter.
